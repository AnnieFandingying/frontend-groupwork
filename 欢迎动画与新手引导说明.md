## 🎯 使用流程

```
1. 点击进入"学习路线"
   ↓
2. 显示欢迎动画（3秒）
   - 转圈加载 + 脉冲光晕
   - "欢迎来到个性化学习路线推荐" (流动渐变文字)
   - "请完成下述量表" (发光脉冲副标题)
   ↓
3. 欢迎屏缩放消失
   ↓
4. 新手引导自动开始（0.5秒后）
   - 透明遮罩 + 背景模糊
   - 高亮聚焦区域（发光边框）
   - 引导气泡（带箭头指示）
   ↓
5. 按照引导填写表单
   - 点击"下一步"继续
   - 点击"上一步"返回
   - 点击遮罩可关闭引导
   ↓
6. 完成表单后点击"生成个性化学习路线"按钮
```

## 🐛 调试

打开浏览器开发者工具（F12），在 Console 中可以看到：
- 🎯 LearningPath mounted
- 📝 Subtitle shown
- 👋 Welcome screen hidden
- 🚀 Starting guide...
- ✨ Starting guide system
- 📍 Updating position for step X
- ✅ Target element found
- ➡️ Next guide: X -> Y

如果引导无法正常进行，检查 Console 是否有错误日志：
- ❌ Target element not found
- ⚠️ Invalid guide step
